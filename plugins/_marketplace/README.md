# VoiceOS Plugin Marketplace

VoiceOS uses the Agent Zero **Plugin Hub** for browsing and installing community plugins.

## Access

- **Web UI**: `python main.py --web` then open http://127.0.0.1:8000 and use Plugin Hub
- **Voice**: "Search plugins for memory", "Install plugin browser"

## Submit a plugin

Add your plugin to the [Agent Zero Plugin Index](https://github.com/agent0ai/a0-plugins).

See `marketplace.yaml` for VoiceOS featured plugins.
